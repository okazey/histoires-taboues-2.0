# Histoires Taboues Skill

Ce dossier contient un skill Claude Code pour automatiser la production d'épisodes documentaires YouTube.

## Installation
Copier le dossier `histoires_taboues_skill` dans ton dossier de travail local.

## Utilisation
Depuis le dossier :

```bash
python scripts/create_episode.py "Béhanzin" "Pourquoi la France a détruit le dernier roi du Dahomey"
```

Claude Code devra ensuite lire `SKILL.md` et remplir progressivement les fichiers générés.
