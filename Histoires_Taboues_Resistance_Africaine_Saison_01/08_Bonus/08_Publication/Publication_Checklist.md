# Checklist de Publication — Bonus / Hors-série

> Dupliquée depuis 00_Guidelines/Checklist_Publication.md.

## Validation contenu
- [ ] Miniature validée (test mobile + A/B)
- [ ] SEO validé (titre, description, tags, chapitres)
- [ ] Description validée
- [ ] Tags validés (15–25)
- [ ] Sous-titres générés (FR + EN)
- [ ] Commentaires épinglés préparés

## Qualité finale
- [ ] Aucune affirmation non sourcée
- [ ] Crédits / sources affichés
- [ ] Audio normalisé (-14 LUFS)
- [ ] Orthographe vérifiée
- [ ] Programmation à l'horaire roadmap

## Après publication
- [ ] Commentaire épinglé
- [ ] Affichage vérifié (mobile + desktop)
- [ ] Partage réseaux
- [ ] Métriques notées
